HTML - The Coding Language for Websites

HTML (Hyper Text Markup Language) is a coding language for creating websites. It uses tags that the computer understands. Webpages are connected through hyperlinks/text (www, https, etc.)

The tags divide the website into elements. These have instructions to let the computer know what to do with the text.
